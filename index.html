Build for All Media

  • Be mindful of all media types, including screens 2″-60″ and paper printouts
    • E.g., avoid using pixel units, let layout elements flow naturally, and test for mobile
  • Utilize the strengths of each browser and work with their weaknesses
    • Corollary: never tell a user to change browsers, make the site work for them
    • E.g., use flexbox design in Chrome now, but fallback with floats and clears for IE8
  • Clients love to see animation, and jQuery makes it just as easy as show/hide
